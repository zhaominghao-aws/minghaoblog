##git的用法

git init 初始化一个本地库
gir clone URL 从远端克隆一个库
git remote add origin URL 与远端库建立连接
git pull 从远端拉取分支并合并到本地
git fetch 是将远程主机的最新内容拉到本地
git add . 将修改添加到工作区
git diff 将修改的内容与工作区对比
git commit -m “” 将已经添加的修改提交到本地库
git push origin 推送到远端库主分支
git push origin/branch 推送到远端库分支
git branch -a 查看所有分支
git checkout 转换到另一个分支，如果没有，创建新的分支
git merge 合并两个分支
git reset HEAD /* 撤销已经add的了的修改
git reset –hard HEAR^ 回到上一个版本
